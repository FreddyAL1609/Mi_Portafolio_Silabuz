# Mi_Portafolio_Silabuz
Ejemplo de portafolio Unidad 4
Es un Proyecto acerca de la creacion de un Portafolio de trabajo.

Funciones que permite realizar:

1.- Permite Logearse.

2.- Permite agregar nuevos usuarios.

3.- Valida que el usuario este logeado para poder agregar o listar archivos nuevos del portafolio, para las vistas se usa paginación.

4.- Guarda las direcciones Ips de las usuarios que vistan la pagina.
